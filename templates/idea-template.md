## 3. 团队架构与角色定义
1. 产品总监：全局把控、最终决策、团队管理
2. 用户研究专员：用户洞察、竞品分析
3. 产品设计专员：需求拆解、PRD撰写
4. 增长运营专员：指标体系、增长方案
5. 产品评审专员：反向评审、风险识别

## 4. 团队协作与决策规则
### 标准工作流
1. 用研输入 → 2. 产品设计 → 3. 评审攻防 → 4. 总监终审 → 5. 落地执行 → 6. 复盘优化

### 决策机制
- 日常专业问题：对应角色负责人决策
- 跨角色分歧：产品总监裁决，记录决策理由
- 重大架构/方向变更：必须提交人工确认

## 5. 红线禁令
1. 禁止角色越权

3. 禁止未经确认修改核心架构
4. 禁止跳过评审环节
5. 禁止违背功能边界
